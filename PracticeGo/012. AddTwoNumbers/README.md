# Add Two Numbers
## Rating: 242
### Description:
Your task is to write a program that adds two given integers, $A$ and $B$, and outputs their sum.

### Input Format
* The first line contains an integer $T$, representing the total number of test cases.
* Each of the following $T$ lines contains two space-separated integers, $A$ and $B$.

### Output Format
For each test case, calculate the sum of $A$ and $B$ and display the result on a new line.

### Constraints
* $1 \leq T \leq 1000$
* $0 \leq A, B \leq 10000$

### Examples:
**Input:**
```
3
1 2
100 200
10 40
```
**Output:**
```
3
300
50
```
**Explaination:**  
* **Testcase 1:** $1 + 2 = 3$. Hence, the first output is `3`.
* **Testcase 2:** $100 + 200 = 300$. Hence, the second output is `300`.
