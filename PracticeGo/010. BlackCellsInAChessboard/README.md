# Black cells in a chessboard
## Rating: 746
### Description:
Given an $n \times n$ chessboard where $n$ is an even number, determine the total number of black cells on the board.

### Input Format
The only line of input contains a single even integer $n$.

### Output Format
Output the total number of black cells in an $n \times n$ chessboard.

### Constraints
* $2 \leq n \leq 100$
* $n$ is even

### Examples:
**Input:**
```
8
```
**Output:**
```
32
```
**Explaination:**  
![image1](https://s3.amazonaws.com/codechef_shared/download/Images/DAANISH/Chess_Board.svg)  
There are 32 black cells and 32 white cells in an 8 × 8 chessboard. So the answer is 32.
